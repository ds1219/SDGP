# Setup and Documentation

## USE BLACK WHEN UPDATING THE PYTHON CODE!

## Written for python 3.11.2

## python modules

```powershell
pip install -r requirements.txt
```

# frontend
start the server using:

## Windows (PowerShell)

```powershell
($env:HTTPS = "true") -and (npm start)
```

## Linux

```bash
HTTPS=true npm start
```
